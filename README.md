# GitHub


## Comandos basicos en consola

* Clonar repositorio
	git clone url del repo

* Estado de nuestros archivos, archivos y la rama
	git status

* Añadir cambios.
	git add .

* Guardar un commit
git commit -m "regañona de Paola"

* Subir cambios
	git pull origin main

* Podemos ver la informacion de los commit
	git log
